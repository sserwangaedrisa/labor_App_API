# Attendance & Payroll Management Web Application

## Overview

The **Attendance & Payroll Management Web Application** is a modern workforce management platform designed to eliminate manual attendance tracking and payroll calculations. The system improves transparency, reduces opportunities for record manipulation, enhances accountability, and promotes professionalism in company operations.

The platform provides a centralized environment where workers, foremen, and company administrators can efficiently manage attendance records, payment requests, and payroll processing.

---

## Problem Statement

Many companies still rely on manual attendance registers and payroll calculations, which can result in:

* Human errors in attendance records
* Inaccurate payroll calculations
* Delayed worker payments
* Lack of transparency
* Record manipulation and fraud
* Time-consuming administrative processes

This project addresses these challenges through automation and role-based workforce management.

---

## Key Objectives

* Eliminate manual attendance tracking
* Automate payroll management processes
* Improve organizational transparency
* Reduce payroll calculation errors
* Enhance workforce accountability
* Provide real-time access to attendance and payment records
* Improve overall professionalism in company management

## 2. User Dashboard System

The platform contains three role-based dashboards:

### Laborer Dashboard

Workers can:

* Log into their accounts securely
* View attendance history
* Track payment history
* Monitor payment statuses
* Access detailed labor cards
* View:

  * Pending work entries
  * Approved work entries
  * Paid work entries

#### Labor Card Features

* Organized attendance records
* Payment tracking
* Work history
* Payroll information

---

### Foreman Dashboard

Foremen can manage workers assigned to their teams.

#### Features

* View assigned workers
* Record daily attendance
* Manage attendance records
* Create payment requests
* Edit attendance information before submission
* Monitor worker performance
* Track payment request status

#### Responsibilities

* Attendance verification
* Payroll request preparation
* Team management
* Workforce monitoring

---

### Owner / Administrator Dashboard

The Owner/Admin dashboard provides complete control over payroll approval and workforce management.

#### Features

* Review submitted payment requests
* Approve payment requests
* Reject payment requests
* Mark payments as completed
* Manage workers
* Monitor workforce operations
* Access company-wide attendance records

#### Automated Notifications

Workers receive email notifications whenever:

* A payment request is approved
* A payment request is rejected
* A payment is processed

* Secure Cookie Management
* Role-Based Access Control (RBAC)
* Protected Routes
* Session Management
* Secure User Authorization

These measures ensure that users only access information relevant to their assigned roles while maintaining a seamless user experience.
